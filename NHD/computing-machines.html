              Disproving the Entscheidungsproblem

              By applying his Turing Machine to the Entscheidungsproblem, Turing encountered the "halting problem," a situation in which his machine could not choose between continuing forever or halting. Using the Incompleteness Theorems, Turing concluded that it was impossible for a Turing Machine to decide; thereby, the Entscheidungsproblem was unsolvable.
              "Turing, like Gödel, looked at computer programs that were analyzing themselves: computer programs can ask questions about computer programs. And what he was able to show is that no computer program can be given, as its input, another computer program and always determine correctly whether that other computer program is going to halt or go into an infinite loop... the system itself is going to have limitations."
              - (Main)
