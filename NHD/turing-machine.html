              The Turing Machine

              Turing's conceptual machine was introduced in his 1936 paper, "On Computable Numbers with an Application to the Entscheidungsproblem."

              There are six basic commands for the Turing Machine, known as the Turing Primitives. These primitives are arranged in a recursive manner, or a configuration, so that a binary number could be computed.
              "The imaginary Turing Machine... did not look like anything that would come to be known as a computer, though in fact it described the fundamental basis of all computing."
              - (Henderson)

              The Universal Machine

              "It is possible to invent a single machine which can be used to compute any computable sequence."
              - Alan Turing
              The Universal Machine is capable of replicating the output of any Turing Machine based on any input, making it the only mathematical machine needed to solve a problem. This machine is considered programmable based on its inputs.

              The Universal Machine forms the framework for computers; allowing them to perform a wide variety of tasks for humans.
              "Turing's Universal machines allow a rudimentary form of programming, and the computers of today still use basic ideas from these machines."
              - (Viehoff)
